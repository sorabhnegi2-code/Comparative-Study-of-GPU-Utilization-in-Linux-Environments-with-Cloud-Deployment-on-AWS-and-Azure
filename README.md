# Comparative-Study-of-GPU-Utilization-in-Linux-Environments-with-Cloud-Deployment-on-AWS-and-Azure
A cloud computing and GPU-focused project demonstrating Linux GPU tools, AWS vs Azure comparison, and virtual machine deployment on Azure.
## 🖥️ Linux GPU Commands Used

- `lshw -C display` → Detect GPU hardware  
- `glxinfo` → Check OpenGL renderer  
- `vulkaninfo` → Verify Vulkan support  
- `glxgears` → Test GPU rendering  
- `vainfo` → Video acceleration info  

---

## ☁️ Cloud Implementation

- Created Virtual Machine on Microsoft Azure  
- Configured Resource Group and IAM  
- Enabled RDP and HTTP access  
- Deployed Web Server
